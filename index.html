Project Overview

This tool helps integrate Betza notation functionality from Fairy-Stockfish into Musketeer-Stockfish for unconventional chess piece movements.

Key Objectives

  • Merge Betza notation parsing and movement generation
  • Preserve existing Musketeer chess functionality
  • Maintain UCI/CECP protocol compatibility
  • Compile with MSVC for Winboard compatibility
  • Generate properly sized executable (~1.5MB)

 Windows Compilation Instructions

                    
To compile the engine for Windows and use with Winboard:

                    

                        

                            
1. Install Visual Studio

                            
Download and install Microsoft Visual Studio Community (free) with C++ development tools

                            
    
                                
  • Visual Studio Community 2019 or later
    • 
                                
  • Make sure to include "Desktop development with C++"
    • 
                                
  • Windows 10/11 SDK
    • 
                            

                        

                        

                            
2. Open Developer Command Prompt

                            
Launch "x64 Native Tools Command Prompt" from Start Menu

                        

                        

                            
3. Clone and Compile

                            

                                git clone https://github.com/ianfab/Musketeer-Stockfish.git
                                cd Musketeer-Stockfish\src
                                copy betza.h types.h position.h (from this integration)
                                copy betza.cpp movegen.cpp (with Betza modifications)
                                nmake -f Makefile COMP=msvc ARCH=x86-64-bmi2 build
                            

                        

                        

                            
4. Winboard Setup

                            
The resulting stockfish.exe will be Windows-compatible:

                            
    
                                
  • Open Winboard → Engine menu → Install New Engine
    • 
                                
  • Browse to stockfish.exe
    • 
                                
  • Set Protocol to "UCI"
    • 
                                
  • Engine should appear as "Stockfish with Betza support"
    •

Troubleshooting

                    

                        

                            
Small executable size (few KB)

                            
Solution: Check linker settings, ensure static linking of runtime libraries

                        

                        

                            
Missing dependencies

                            
Solution: Install Visual C++ Redistributable or use static linking

                        

                        

                            
Compilation errors

                            
Solution: Check MSVC version compatibility, update Windows SDK
